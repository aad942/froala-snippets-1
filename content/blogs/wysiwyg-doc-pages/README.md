# A Simple Developer Docs Demo

This demonstrates how to use Froala WYSIWYG editor to create documentation pages for internal tools or software products and services.

---

## Features

- **Add New Pages**: Create new documentation pages with a title and some HTML content.
- **View and Update Existing Pages**: Update the title and content of existing pages using a WYSIWYG editor.
- **Rich-text Editing**: Apply different formatting and insert rich media to page contents using the editor.

---

## Prerequisites

To run this example, you will need:

1. Froala Editor

---

## How to Run

### 1. Get the files from this repository and paste them into your project folder.

### 2. Create a database table that matches the fields in the PHP files.

### 3. Run the application on your browser.

---

### How It Works

1. Click the "Add Page" button in the sidebar.
2. Add a title for the page using the text input and some HTML content using the editor. Click "Save."
3. Navigate to created pages to view and edit them or create more pages.

---

### Notes

- Experiment with other features like spelling and grammar checking, code view, and AI integration, which you can include via plugins.
- Try storing different content (Markdown, HTML, media files) separately.
