# Plain Text to HTML Using Froala Editor

This shows how you can paste or type plain text content onto a WYSIWYG editor and have it convert the content into HTML.

---

## Features

- **Plain Text Pasting**: Paste plain text from regular text editors.
- **Code View**: Convert pasted or typed plain text content into HTML.
- **Code Beautification**: In code view, format generated code to look better.

---

## Prerequisites

To run this example, you will need:

1. Froala Editor

---

## How to Run

### 1. Get the files from this repository and paste them into your project folder.

### 2. Run the application on your browser.

---

### How It Works

1. Paste or type plain text into the editor.
2. Click the "Code View" button on the toolbar.
3. Wait for the editor to show the equivalent HTML.

---

### Notes

- You can also try using other libraries or custom logic to process more types of complex plain text content (number sign for headings, etc.)
